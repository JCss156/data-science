# COVID-19 Data Analysis: United States vs South Africa

This repository contains a data analysis study based on the **Our World in Data COVID-19 dataset**, with a focus on comparing the pandemic evolution in the **United States** and **South Africa**.

The project was developed as a practical data manipulation and visualization exercise using Python and Jupyter Notebook. Its main goal is to explore how COVID-19 evolved across different countries and to highlight differences in infection, vaccination, and mortality patterns.

## Project goals

- Practice data manipulation with real-world data
- Explore and compare pandemic indicators across countries
- Visualize differences in vaccination, cases, and deaths
- Build an initial analytical workflow using Python for data science studies

## Main analysis

The notebook explores pandemic-related indicators from the OWID dataset, with emphasis on comparisons such as:

- **United States vs South Africa**
  - new cases per million
  - total deaths per million
  - general pandemic progression over time

It also includes additional exploratory comparisons involving other countries in some plots.

One of the main observations from this study is that, in the selected comparison, the **United States appears to have been more heavily affected by the Omicron wave than South Africa**, even with higher vaccination coverage.
